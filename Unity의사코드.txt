경마게임

맵 구성 :
1. 달리기를 할 트랙
2. 1,2,3등을 위한 단상

게임플레이 : 
GameObject 5개를 랜덤한 속도로 전진하게 하여,
결승점에 도달할 경우,
게임을 끝내고 1,2,3등은 1,2,3등 단상에 각각 위치 시킨다.

논리 세분화(나) : 
1. 게임 오브젝트를 5개 만든다.
2. 시작 위치는 0으로 한다.
3. 매 초마다 이동할 랜덤한 숫자(게임 오브젝트의 속도가 될 숫자)를 각 오브젝트에 부여한다.
4. 결승점의 값을 설정한다.
5. 결승점의 값을 초과한 오브젝트에 1,2,3순위를 메겨 단상에 위치시킨다.

논리 세분화(강사님) : 
1. GameObject 5개를 만든다.
2. GameObject 5개를 동시에 전진시킨다.
3. 전진시에 각 GameObject는 랜덤한 속도로 움직인다.
4. 전진시에 GameObject의 위치가 결승점을 넘었을 경우 등수를 메긴다.
5. GameObject 5개가 모두 결승점을 넘었을 경우 달리기를 중단한다.
6. GameObject 중 1,2,3등을 한 GameObject는 각각 단상의 1,2,3등에 위치시킨다.

의사코드(나) : 
시작지점 = 0
랜덤 난수 변수 생성
매 프레임마다 전진할 게임 오브젝트 5마리에 속도가 될 랜덤변수를 넣어준다

의사코드(강사님) : 
GameObject  리스트에 GameObject 5개 대입
GameObject 등수 리스트 생성

if(게임 진행중)
{

		if(GameObject.Position > 결승점)

			등수 리스트.Add(GameObject)
		}

	if(등수리스트.count > 5)

		}
